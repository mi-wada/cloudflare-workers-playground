# urlshortener

## Development

### Prerequisites

- Node.js
- npm

### Deploy

```shell
npm run deploy
```

### Run locally

```shell
npm run dev
```

### Run tests

```shell
npm run test
```

### Run lint

Without fixing issues:

```shell
npm run lint
```

With fixing issues:

```shell
npm run lint:fix
```

### DB Migration

Local:

```shell
npm run db:migrate:local
```

Production:

```shell
npm run db:migrate:prod
```
